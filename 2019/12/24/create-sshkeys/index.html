MacOS生成SSH Keys - 简单版
2019-12-24|SSH|

ssh keys 的应用场景

ssh keys 通常用于以下几个场景：

  • Github 仓库的身份认证
  • 远程服务器的身份认证
  • 其他…

    ssh keys 的存储位置

    linux \ mac os 存储位置

    一般在 ~/.ssh 路径下

    windows 存储位置

    一般在 c:\Users\administrator.ssh 路径下

    ssh keys 的构成

    ssh keys 分为两个文件
    公钥：id_rsa.pub
    私钥：id_rsa
    以上名称可以在生成秘钥文件时指定，默认是 id_rsa 命名。
    关于公钥和私钥加密算法的原理，可以自行查阅相关资料。

    ssh keys 的生成

    ssh-keygen -t rsa -C “你的邮箱”
    剩下的可以一路回车就可以生成秘钥文件到上面说的各平台的存储目录，也可以根据提示设置密码，更改存储位置。
