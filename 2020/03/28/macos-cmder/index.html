MacOS常用命令
2020-03-28|MacOS技巧|
  • 持续更新

lsof

查看端口占用

  • lsof -i:1080

    查看占用1080端口的应用

  • lsof -i tcp:1080

    查看占用1080端口的tcp协议的应用

launchctl

查看服务列表

  • launchctl list

    查看所有服务

  • launchctl list | grep xxx

    使用 lsof 查出占用端口的pid

    再使用 launchctl 配合 grep 查看指定pid，筛选出占用端口的具体服务是哪个

活动监视器

通过活动监视器找到服务相关文件

通过 lsof -i 或者 launchctl 找到相关服务的 pid

然后通过 活动监视器找到对应pid或服务名称，定位到这个服务

双击活动监视器中显示的进程，然后可以在 打开的文件和端口 看到相关的进程和端口

brew

常用命令

  • 搜索软件

    brew search xxx

  • 安装软件

    brew install xxx

  • 安装未在应用商店上架的软件

    brew cask install xxx

du

查看层级目录大小

  • 指定查看的层级

    du flutter/bin/ –max-depth=1 -h

  • 直接查看指定目录大小

    du flutter -sh

netstat

查看端口占用

  • 查看所有端口占用

    netstat -ntlp

finder 输入路径

  • 快速打开指定路径

    快捷键：Shift + Command + G
