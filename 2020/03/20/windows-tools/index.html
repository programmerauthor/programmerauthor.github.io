windows实用工具
2020-03-20|Windows工具|

ProcessExplorer

  • 微软推出的Windows超级任务管理器

Everything

  • Windows下的超级搜索

Electerm

  • Electron编写的跨平台 terminal/ssh/sftp 管理工具（支持 linux,mac,win）

Vysor

  • Android 设备 PC 端操作工具，可以通过adb，实现在PC上操作Android设备
  • 拥有Windows安装版和Chrome跨平台插件

Scrcpy

  • 跟Vysor类似的 Android设备 PC 端操作工具，实现在PC上操作Android设备
  • 优点：免费，功能强大，跨平台
  • 缺点：操控舒适性上较Vysor有所欠缺

Cmder

  • Windows下比自带cmd和PowerShell好用一百倍的命令行工具
  • 安装自带Git的版本更好用

Typora

  • Markdown文本编辑器，非常好用，跨平台（Linux，Mac，Windows）
  • 可以导出Html、PDF、Word等，功能非常强大

Haroopad

  • 另外一款好用的 Markdown文本编辑器
  • 在发现Typora之前选它，之后 Typora

Wiztree

  • Windows下非常强大高效率的扫描文件夹大小的工具

MindMaster

  • 好用的思维导图软件
  • 缺点：需要收费（终身全平台激活¥680）

Nomachine

  • 非常好用的远程桌面管理软件
  • 传说中10K网速即可连接（实际上是很快，但是没这么夸张）
  • 优点：速度快、免费、质量好
  • 缺点：远程连接需要手动搭建中转节点
  • 跨平台（Linux，Mac，Windows）

TeamViewer

  • 另一款非常好用的远程桌面管理软件
  • 优点：非常强大，性能好，速度快，可以直接远程连接，跨几乎所有平台
  • 缺点：收费，很贵，免费经常被以商业使用为由阻断

VNCServer/VNCViewr

  • 很强大的远程连接软件
  • 优点：开源、免费、跨平台（Linux，Mac, Windows）
  • 缺点：速度有点慢

VLC

  • 非常强大的跨平台播放器（Linux，Mac, Windows，各种移动端）

Synergy

  • 非常强大的共享鼠标键盘工具
  • 可以使用一套键盘鼠标在多台主机上无缝跨主机跨屏幕操作

PostMan

  • 接口模拟请求测试工具
  • 模拟Post/Get进行请求，测试接口

Scoop

  • 包管理工具，类似linux的apt，macos的brew（跟Chocolatey类似，推荐使用Scoop）
  • 优点：scoop自由度很高，可以将软件部署到任意盘中
  • 缺点：scoop诞生时间比较短，包的丰富度不是很高

Chocolatey

  • 包管理工具，类似linux的apt，macos的brew（跟Scoop类似，推荐使用Scoop）
  • 优点 : chocolatey诞生时间比较长，包的丰富度比较高
  • 缺点 ：相对比较不灵活，软件安装路径默认在C盘

Meld

  • 文件对比工具
  • 强大的跨平台（支持Linux、Mac、Windows）问价你对比工具，可以用来合并代码

ScreenToGif

  • 一款Windows平台非常强大的截取Gif图的工具
  • 可以录制屏幕、录制摄像头、画板录制甚至制作空白动画等

uTools

  • 非常强大的跨平台聚合工具平台
  • 用 Electron 编写，支持 Windows、MacOS、Linux

Hexyl

  • 非常小巧的面向终端的十六进制查看器
  • 支持跨 Windows、MacOS、Linux
  • windows上安装依赖于 Rust 的 cargo
