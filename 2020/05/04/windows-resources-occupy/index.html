Windows找出文件占用的根源
2020-05-04|Windows技巧|

很多人都会碰到删除程序时，碰到过 操作无法完成，因为其中的文件夹或文件已在另一程序中打开 场景。

下面说一下如何找到是什么在占用这个文件夹。

  • 首先 Ctrl+Alt+Del 打开任务管理器，选择 性能 选项卡

  • 然后点击底部的 打开资源监视器

  • 选择 CPU 选项卡

  • 在 关联的句柄 模块 搜索句柄 输入框中输入被占用文件的名字搜索

  • 等待一会儿出现搜索结果，就找到了占用文件夹的真正程序

  • 选中这个占用程序，右键选择结束进程即可

  • 结束掉这个占用进程，再回去删除文件夹，发现已经可以删除了
