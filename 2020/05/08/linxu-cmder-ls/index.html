每日一条Linux命令：ls
2020-05-08|Linux命令|

简介

ls 是 unix 类系统中最常用的一个命令，但是很多朋友可能并不清楚它的很多方便用法，例如按修改日期排序等。本文演示 ls 最常用的实用方法。

常用方法

每行列出一个文件

ls -1

列出所有文件，包括隐藏文件

ls -a

所有文件的长格式列表（权限，所有权，大小和修改日期）

ls -la

长格式列表，其大小使用人类可读单位显示（KB，MB，GB）

ls -lh

长格式列表按大小排序（降序）

ls -lS

所有文件的长格式列表，按修改日期排序（最旧的优先）

ls -ltr
